### 作业

**操作步骤：**
```
1.组长给你一个git仓库地址 （里面有你要开发的项目）
2.将代码克隆到本地 git clone 地址
3.组长添加你为协作了
4.本地仓库添加远程仓库地址）
5.修改代码
6.提交之前一定要pull 拉去最新的代码
7.提交到本地仓库 在提交到远程仓库
注意：可能会冲突 共同修改了同一个文件
```

```
组长自行创建一个 脚手架 在master分支，vue3.0脚手架，（三方插件可以一点点完事）
组长组织会议，共同来研讨开发过程可能出现得问题，或者潜在的问题，然后一起制定开发计划
```
